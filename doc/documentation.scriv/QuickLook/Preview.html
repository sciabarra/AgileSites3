    • Installer
      Before starting the installer, you need to run a “collect” task that will prepare the repo/local folder, copying in repo/bin the sbt-launcher, and configuring it, and checking out the “bigbang” folder

      The installer done with izpack will then pack the folder the following directories

      bigbang
      repo

      It will also create the bigbang/agilesites.{sh,cmd} scripts

      It will finally ask if you want to download and install Sites, and if yes will run the sitesInstall command
