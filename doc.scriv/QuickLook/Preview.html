          • Installation
            Download the agilesites-bigbang.zip from www.agilesites.org/bigbang.zip and unzip it.

            Copy agilesites.properties.dist into agilesites.properties.

            Edit it to match your system. Most important you have to set:

            • java: java executable you want to use
            • site.focus : the name of the main site of your project. It is recommended it is a lower case name
            • sites.directory is an empty writable directory where to install sites

          • Create a new site
            The first step to implement a new site with AgileSites is to create a new site.

            You do that creating a class and adding an annotation

            Then

          • 3 Install Sites
            Open either the terminal or the command prompt in the big bang project folder and type sbt. You should see something like this:

            >>> agilesites.properties
            sites.directory=/data/sites/bigbang
            sites.focus=bigbang
            [info] Set current project to bigbang
            (in build file:../bigbang/)


            Now at the command prompt type

            bigbang> sitesInstall

            You will see a lot of output while it will install Sites.

            Now you can install AgileSites with the command:


            bigbang> asSetup

            That’s it. You have AgileSites insta...

          • 4 Configure Eclipse
            1) add in the file project/plugins.sbt with the following lines (separated by an empty line from the preceding):

            addSbtPlugin("com.typesafe.sbteclipse" % "sbteclipse-plugin" % "3.0.0") 

            2) add in the file build.sbt the following line:

            EclipseKeys.projectFlavor := EclipseProjectFlavor.Java 
            3) type the following to generate an Eclipse project:

            bigbang> ; reload ; eclipse
