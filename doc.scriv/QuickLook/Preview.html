          How to embed sbt and run it offline with multiple profiles

          In

          setup/bin/sbt.sh put a script to launch SBT with the following options

          JAVA_HOME
          IVY_REPO
          PROFILE

          Then create in setup a script

          Cd here
          JAVA_HOME=…
          IVY_REPO=…
          PROFILE=…
          Call bin\sbt.cmd %*
